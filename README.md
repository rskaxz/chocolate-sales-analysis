# CHOCOLATE SALES ANALYSIS

# > Data Analytics Portfolio Project

# Dataset source: Chocolate Sales Dataset 2023 - 2024

# \---

# \## 📌 Business Task

# Identify key factors influencing revenue and profitability in a global chocolate retail business, and deliver data-driven recommendations to management regarding product strategy, store performance, and customer behavior to improve overall profit margin

# \---

# \## Key Business Questions

# •	Which products contribute the most to total revenue? 

# •	Which stores have the highest and lowest performance? 

# •	How do sales trends evolve over time (monthly or yearly)? 

# •	Who are the most valuable customers? 

# •	What is the Average Order Value (AOV)?

# \---

# \## Stakeholders

# •	Marketing Team → to optimize promotional strategies 

# •	Sales Manager → to improve sales performance 

# •	Management → to support strategic decision-making

# \---

# \## 📁 Dataset Overview 

# | Table | Rows | Description |

# |---|---|---| 

# | sales.csv | 1,000,000 | Transaction fact table | 

# | products.csv | 200 | Product dimension | 

# | customers.csv | 50,000 | Customer dimension | 

# | stores.csv | 100 | Store dimension | 

# | calendar.csv | 731 | Date dimension |

# &#x20;\*\*Schema:\*\* Star schema (1 fact + 4 dimensions) 

# \*\*Period:\*\* 2023–2024 

# \---

# Tools Used

# •	SQL (BigQuery)  data cleaning, exploration, data querying, filtering

# •	Power BI  dashboard creation and interactive

# \---

# \## 🔍 Key Findings 

# •	\*\*Total Revenue: \*\* $25,486,129 across 1M transactions 

# •	\*\*Total Profit: \*\* $10,194,565 (avg margin 40.00%) 

# •	\*\*Top Brand by Volume: \*\* Ferrero (183,603 orders) 

# •	\*\*Largest Market: \*\* Canada ($5.08M revenue) 

# •	\*\*Largest Customer Segment: \*\* Boomer 50+ (397K orders, 39.7%) 

# •	\*\*63% of orders \*\* placed at full price — no discount dependency 

# •	\*\*Data Quality Note: \*\* Profit margin is artificially fixed at exactly 40% across all segments, indicating a synthetic dataset. Analysis therefore focuses on volume, geographic, and demographic distribution rather than profitability drivers.
